# Technology Stack

This project leverages a modern, high-performance tech stack designed for real-time collaboration and AI-powered productivity.

## Core Framework
- **[Next.js 14](https://nextjs.org/)**: React framework for building server-side rendered and statically generated web applications.
- **[TypeScript](https://www.typescriptlang.org/)**: Type-safe JavaScript for improved developer experience and code quality.

## Real-time Collaboration
- **[Liveblocks](https://liveblocks.io/)**: Infrastructure for building collaborative features like presence, storage, and real-time syncing.
- **[Yjs](https://yjs.dev/)**: Shared data types for building collaborative applications.

## AI Integration
- **[Google Gemini AI](https://ai.google.dev/)**: Powering intelligent features like chat-to-document and automated translation.

## Authentication & Security
- **[Clerk](https://clerk.com/)**: Comprehensive user management and authentication solution.
- **[NextAuth.js](https://next-auth.js.org/)**: Flexible authentication for Next.js (Beta).

## UI/UX
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework for rapid UI development.
- **[Shadcn UI](https://ui.shadcn.com/)**: Reusable components built with Radix UI and Tailwind CSS.
- **[Lucide React](https://lucide.dev/)**: Beautiful, consistent icons.
- **[Framer Motion](https://www.framer.com/motion/)**: Production-ready motion library for React.
- **[BlockNote](https://www.blocknotejs.org/)**: A modern block-based rich-text editor.

## Backend & Database
- **[MongoDB](https://www.mongodb.com/)**: Document-oriented NoSQL database for data persistence.
- **[Mongoose](https://mongoosejs.com/)**: Elegant MongoDB object modeling for Node.js.

## State Management
- **[Zustand](https://docs.pmnd.rs/zustand/getting-started/introduction)**: A small, fast, and scalable bearbones state-management solution.
- **[TanStack Query (React Query)](https://tanstack.com/query/latest)**: Powerful asynchronous state management for TS/JS.
